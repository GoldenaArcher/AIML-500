# AIML-501 Assignment 5: Pre-Trained Model Decision Matrix

## Explanatory Document

This artifact compares pre-trained models across NLP, computer vision, and tabular data to show how model choice depends on balancing size, accuracy, speed, and explainability rather than chasing a single benchmark. The matrix highlights that BERT remains a strong baseline for language understanding, DistilBERT offers a better speed-size trade-off for lighter NLP use cases, MobileNetV2 favors lightweight vision deployment, EfficientNetB0 trades a bit more compute for stronger vision accuracy, and TabPFN gives a fast-start option for small tabular tasks even though it is less interpretable than tree-based methods.

I designed the page as a decision matrix because this format makes trade-offs easier to compare quickly across domains. The layout keeps the methodology, matrix, recommendations, and conclusion separate so the artifact works both as a visual portfolio piece and as a practical reference for model selection discussions.
