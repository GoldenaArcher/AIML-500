# AIML-501 Assignment 1: Machine Learning Algorithm Framework

I put this together for classmates and other beginners who just need a simple way to compare common machine learning algorithms. I tried to keep it easy to scan, so I used a visual layout and a matrix instead of long explanations. Adding real examples helped make the differences feel more practical.

What stood out to me while doing this assignment was how much easier it is to understand these algorithms when they are all in one place. Looking at them side by side helped me sort out where each one fits, especially across tabular data, computer vision, NLP, and generative AI. It also made me pay more attention to how I explain technical ideas, not just whether the information is correct.

Most of my revisions were around clarity. I cleaned up the references, added source tags for each algorithm, and separated actual sources from the tools I used to build the page. I have not received instructor feedback yet, so the changes so far mostly came from reviewing the artifact myself and fixing parts that felt vague.
