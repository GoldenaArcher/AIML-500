# AIML-501 Assignment 1: Machine Learning Algorithm Framework

I put this together for classmates and other beginners who need a simple way to compare common machine learning algorithms. When the information is dense but still manageable, I think a tabular format is one of the clearest ways to compare differences, so I used a matrix as the core of this artifact instead of relying on long paragraphs. Adding example use cases also helped make the differences more practical.

This assignment also fits well with how I usually like to organize technical information. Since the requirement was to present 8 to 10 algorithms and explain their types, application domains, use cases, and brief descriptions, a tabular format felt much cleaner than listing them one by one. It made the comparisons easier to read and helped me maintain a consistent structure across all the algorithms. In some previous artifacts, I tried a tree-style layout, but that format did not work as well for this kind of side-by-side comparison. Unless the tree is interactive and can expand with more detail, the fixed space limits how much information can be shown clearly.

Most of my revisions were around clarity. I cleaned up the references, added source tags for each algorithm, and separated actual sources from the tools I used to build the page. I also reworded a few areas that felt too broad so the final version matched more closely with how I wanted the comparison to read.
